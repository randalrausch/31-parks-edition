# Support

Thanks for playing 31 · National Parks Edition! Here's where to go:

- **Just want to play?** → [play31.fun](https://play31.fun) — no install, no
  account.
- **Something's broken?** → open a [bug report](https://github.com/randalrausch/31-parks-edition/issues/new?template=bug_report.yml).
  Include your mode (solo / pass-and-play / online), the version from the in-app
  **About** dialog, and any browser-console errors.
- **Have an idea or a question?** → start a
  [Discussion](https://github.com/randalrausch/31-parks-edition/discussions), or
  open a [feature request](https://github.com/randalrausch/31-parks-edition/issues/new?template=feature_request.yml).
- **Want to add a park or some art?** → the friendliest way in. See
  [docs/THEMES.md](docs/THEMES.md) and the [ROADMAP](ROADMAP.md).
- **Found a security issue?** → please report it privately, not as a public
  issue — see [SECURITY.md](SECURITY.md).
- **Want to contribute code?** → start with [CONTRIBUTING.md](CONTRIBUTING.md).

This is a hobby project maintained in spare time, so please be patient with
response times. 🌲
